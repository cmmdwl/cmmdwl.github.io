硬件密码键盘设计

发表于 2023-08-01
本文字数： 408 阅读时长 ≈ 1 分钟

起因

记住一大串密码实在是太头疼了，于是，我希望能得到一份可以简单代替我的记住密码的工具，需要支持这些东西

  1. 必须是纯硬件的
  2. 必须有办法导出密码避免密码丢失
  3. 必须有带有一个开机密码且不能定死密码
  4. 需要能够模拟usb键盘进行输入且可以当蓝牙键盘
  5. 最好还能在多次错误密码的情况下丢失密匙或有个丢失密码
  6. 需要能抗暴力拆解

基于综上设计，我选择了ESP32-S3 通过ESP-IDF进行编译
为什么使用ESP-IDF进行编译呢？

  • ESP-IDF相对于Arduino更将安全，基于Boot Load 的可信启动可以防止被刷flash，并且可以关闭所有的调试渠道防止被物理破解
  • espidf基于c进行编译，相比Arduino会更加快捷

选型完成后就是我们应该找那些东西了

  1. 有个键盘，最好是屏幕加触摸
  2. 有个可信的RAM，最好是掉电消失的

所有可能需要的东西都列在上面了，剩下就是基于开发板进行编辑的产品了

低电压下无人机速度会变慢

发表于 2020-12-03 更新于 2023-08-01
本文字数： 414 阅读时长 ≈ 1 分钟

这几天整了个大疆的御air2来进行飞行，60电拉了3km/120H以外，返航电量后开始往回飞行，越飞感觉越不对，飞机的速度从20%时候就开始降下来了，从原来的60变成38左右，猜测，可能是因为飞机电池已经不太够提供满速飞行返回时间了，于是我开始拉低高度杆，给飞机来了个0.5m/s的下降速率，飞机的速度就上升的42km/h最终有惊无险的返回了我所在的位置的旁边，结果最终由于飞机起飞高度有30m左右为飞机带来了更远的返回距离，最终电量还剩下3%电池电压也变成了3节3.65，最终我下楼去人行横道上捡回了我的air2，最后给大家提个醒，大疆的低电量返航是按距离算的，如果你还有高度话那么高度能为你提供不少而外的电量，如果没有，在提示返回时候请立马开始运动挡返回，如果有高度话，例如我的4km500H/满速降高返回后还能够剩个20%左右的电量，但是为了电池的安全，建议充电到80%，放电到20%时，就请充电吧。还有，一定要留有电池余量的，自己心里一定要留点数，大疆有进度条，其他飞机请自测一下全功率放电到标称3.7时候有多久，自己拉锯时候打一个40%的定时器，到点就回来，省得报废电池或者飞机。

前言
今天给我的设备进行清灰，结果发现了一个神奇的问题

自从我把显卡从主板上拿下来擦擦灰后，插回去就没工作过，VGA白色灯亮，设备一长鸣两短鸣。

我的设备配置是

主板：华硕B360-F
显卡：迪兰恒进RX580-2048SP，
CPU：I7-8700
内存：威刚万紫千红DDR4-3200/16GB

既然显卡接触没问题，或许插口出现了点问题呢？

换插座！

结果失败：每个插座都是同样的报错声音

那么会不会是显卡玄学炸了？

然后发现，我所做的一切，都没有换过设备
